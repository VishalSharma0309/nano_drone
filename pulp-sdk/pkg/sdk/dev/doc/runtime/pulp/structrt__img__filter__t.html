Description

This structure is used to define the HW filter coefficients (red, green, blue). These coefficients are applied as follows: pixel = red*r_coeff + green*g_coeff + blue*b_coeff

For example, with the RGB565 format, each pixel is coded in 2 bytes: red = pixel[15:11] green = pixel[10:5] blue = pixel[4:0] In the rx buffer, each pixel received would be the result of the above formula.
