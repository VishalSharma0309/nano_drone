Description

This structure is used by the runtime to manage a cluster remote copy with the HyperRAM. It must be instantiated once for each copy and must be kept alive until the copy is finished. It can be instantiated as a normal variable, for example as a global variable, a local one on the stack, or through a memory allocator.
