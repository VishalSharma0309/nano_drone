◆ rt_gpio_configure()

int rt_gpio_configure ( uint8_t  group,
uint32_t  mask,
rt_gpio_conf_e  flags 
)

This function is used to configure a set of GPIO. The specified configuration is applied to the set of GPIOs. Can only be called from fabric-controller side.

Parameters
groupGPIO group. Must always be 0 for now.
maskA mask of GPIOs to be configured. There is one bit per GPIO, bit 0 is GPIO 0 and bit 31 GPIO 31. The configuration will be set for all GPIOs which have their corresponding bit set to 1.
flagsThe configuration to be applied to the set of GPIOs. Check the description of rt_gpio_conf_e for further information.
Returns
0 if the operation is successfull, 1 otherwise

is_outA flag for choosing the direction of the gpio
