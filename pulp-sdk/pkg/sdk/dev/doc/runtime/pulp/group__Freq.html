◆ rt_freq_domain_e

enum rt_freq_domain_e

Describes a frequency domain.

Enumerator
RT_FREQ_DOMAIN_FC 

Frequency domain for fabric controller.

RT_FREQ_DOMAIN_CL 

Frequency domain for cluster 0. Other cluster domains can be obtained by adding the cluster ID to this constant.

RT_FREQ_DOMAIN_PERIPH 

Frequency domain for peripherals with secondary udma clk

RT_FREQ_DOMAIN_PM 

Frequency domain for power manager.
