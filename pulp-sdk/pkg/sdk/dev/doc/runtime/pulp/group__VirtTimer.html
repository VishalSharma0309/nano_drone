◆ rt_timer_flags_e

enum rt_timer_flags_e

This is used to describe hos to configure a virtual timer.

Enumerator
RT_TIMER_PERIODIC 

Periodic mode. The timer will trigger a function call periodically.

RT_TIMER_ONE_SHOT 

One-shot mode. The timer will trigger a function call only once and need to be re-armed.

◆ rt_timer_create()

int rt_timer_create ( rt_timer_t *  timer,
rt_timer_flags_e  flags,
rt_event_t *  event 
)

A virtual timer can be created in order to trigger function callbacks based on time. These callbacks can be triggered either periodically or once.

Parameters
timerA pointer to the timer descriptor, which must be allocated by the caller. The descriptor must remain valid until the timer is destroyed.
flagsThe flags for configuring the timer.
eventThe event. This must be used to describe the function callback called by the timer. This event cannot be NULL and cannot be a blocking event.
Returns
0 if the operation is successfull, -1 otherwise.

◆ rt_timer_start()

void rt_timer_start ( rt_timer_t *  timer,
int  time_us 
)

This will start triggering function callback calls. If the timer is configured periodically, the call will be triggered everytime the specified amount of time has passed. If it is configured in one-shot mode, a call is triggered only once and the timer must be re-armed using this function.

Parameters
timerA pointer to the timer descriptor (the one provided when the timer was created).
time_usThe time in microseconds after which the function callback is called.
